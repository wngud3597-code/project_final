# 데이터 출처와 이용 표시

## 관광정보

- 제공기관: 한국관광공사
- 데이터: 국문 관광정보 서비스(TourAPI 4.0)
- 원본 파일의 라이선스 안내: 공공누리 제3유형
- 서비스 화면에서 출처를 상시 표시합니다.
- 원본 JSON의 필드 값은 수정하지 않습니다.
- 이미지별 `cpyrhtDivCd` 값을 상세 화면에 표시합니다.

## 날씨

- 제공기관: 기상청
- 데이터: 단기예보 조회서비스
- 사용 기능: 초단기실황조회, 초단기예보조회
- 공공데이터포털 이용허락: 공공누리 제1유형(출처표시)

## 지도

- 지도 데이터 및 타일: © OpenStreetMap contributors
- 지도 화면 하단에 출처를 표시합니다.
- 표준 타일 서버는 개발·시연 규모에서만 사용하며 대규모 운영 시 별도 타일 제공자를 사용해야 합니다.
